TabActionbar
============
Actionbar 切换  滑动actionbar菜单  

关键词：swipeActionbar slidingActionbar  tabactionbar   可滑动的actionbar  actionbar Tab滑动

这个示例工程，将actionbar上实现tab的动态切换，同时兼容8以上所有平台  项目依赖v7 appcompat 包 请自行添加 

有问题 欢迎fork！
